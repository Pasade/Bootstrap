Bootstrap

Bootstrap basa su diseño en un sistema de columnas, divide toda la página en 12 columnas, la ventaja que ofrece Bootstrap es que sin necesidad de usar los Media Querys podemos indicar el numero de columnas que debe emplear segun el dispoistivo que acceda a la página.

Los Media Querys ya vienen definidos en Bootstrap por eso nosotros no los vamos a usar, a continuación mostraremos los Media Querys utilizados por Bootstrap.

Dispositivos peuqeños Como móviles, anchura menor a 768ppx

@media (min-width: @screen-xs-min){...}

Dispositivos pequeños como tablets, anchura mayor o igual a 768px

@media (min-width: @screen-sm-min){...}

Dispositivos medianos como ordenadores, anchura mayor o igual a 992px

@media (min-width: @screen-md-min){...}

Dispositivos grandes como ordenadores, anchura mayor o igual a 1200px

@media (min-width: @screen-lg-min){...}

En ocasiones, también se utilizan las siguientes media queries que definen la propiedad max-width y permiten restringir los dispositivos a los que se aplican los estilos CSS:

@media (max-width: @screen-xs-max) { ... }

@media (min-width: @screen-sm-min) and (max-width: @screen-sm-max) { ... }

@media (min-width: @screen-md-min) and (max-width: @screen-md-max) { ... }

@media (min-width: @screen-lg-min) { ... }
