Como preparar todo para trabajar con Bootstrap

Descargar Archivos y Preparar Carpetas

Lo primero que debemos hacer es dirigirnos a la página de bootstrap y descargarnoslo.

Ir a Bootstrap.

También debemos descargar JQuery ya que Bootstrap trabaja con JQuery.

Ir a JQuery

Descomprimimos el rar de bootstrap y nos quedarán 3 carpetas, CSS, JS y fonts.

Copiamos el archivo descargado de JQuery en la carpeta de js.

Ahora ya tenemos todos los archivos ncesarios y las carpetas posicionadas para enpezar a trabajar en nuestro código.

Enlaces a los CSS y JS

Ahora debemos enlazar en el head de nuestra página los estilos CSS y los archivos JS

Antes de nada debemos agregar el "meta Viewport"

<meta name="viewport" content="width=device-width, user-scalable=no, initial-scale = 1.0, maximum-scale = 1.0, minimum-scale = 1.0">

Esta parte es muy importante y no debemos olvidarnos nunca de agregar nuestro meta viewport, para que podamos trabajar con diseños responsive.

Ahora debemos enlazar los estilos CSS, el archivo que vamos a enlazar se encuentra dentro de la carpeta CSS y se llama "bootstrap.min.css"

<link rel="stylesheet" type="text/css" href="css/bootstrap.min.css">

Ahora debemos agregar los enlaces a nuestros archivos JS, el primero que agregaremos sera el archivo JQuery y posteriormente el de Bootstrap, en ese orden. Si vamos a añadir algun archivo con nuestros propios estilos CSS o algun archivo JS, estos deben colocarse despues de los que vamos a meter ahora aquí.

Quedaría asi:

<script type="text/javascript" src="js/jquery-3.2.1.min.js"></script> <script type="text/javascript" src="js/bootstrap.min.js"></script>

Así es como debería quedar todo nuestro Header:

<!DOCTYPE html> <html lang="es"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, user-scalable=no, initial-scale = 1.0, maximum-scale = 1.0, minimum-scale = 1.0"> <link rel="stylesheet" type="text/css" href="css/bootstrap.min.css"> <script type="text/javascript" src="js/jquery-3.2.1.min.js"></script> <script type="text/javascript" src="js/bootstrap.min.js"></script> <title>bootstrap</title> </head> <body> </body> </html>

Y bien, ya tenemos todo listo para empezar a trabajar con Bootstrap.
